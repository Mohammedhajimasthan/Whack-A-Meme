# 😂 Whack-A-Meme

A fast-paced browser game inspired by the classic "Whack-A-Mole" concept, where players test their reflexes by clicking randomly appearing memes before they disappear.

---

## 🎮 Project Overview

Whack-A-Meme is an entertaining and interactive web-based game designed to improve reaction speed, hand-eye coordination, and user engagement through humorous meme-based gameplay.

Players must quickly identify and click appearing memes to earn points while competing against the clock to achieve the highest score possible.

## 🚀 How to Play

1. Start the game.
2. Watch for memes appearing on the screen.
3. Click the meme before it disappears.
4. Earn points for every successful hit.
5. Continue until the timer ends.
6. Try to achieve your highest score.
